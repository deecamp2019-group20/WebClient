斗地主游戏，后端基于Python+Tornado+MySQL开发，前端 Phaser 引擎




Quick Start

    git clone https://github.com/mailgyc/doudizhu
    cd doudizhu
    mysql --user=root -p < schema.sql
    pip3 install -r requirements.txt
    cd doudizhu
    python3 app.py
    Now visit http://127.0.0.1:8080
